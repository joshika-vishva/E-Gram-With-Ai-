# e-Gram Digital Platform

Digital platform that allows villagers to report problems, receive announcements, access government services, and communicate with local authorities.

## Features
- **Zara AI Assistant**: Dedicated agricultural AI powered by Gemini 2.5 Flash.
- **Bilingual Support**: Full support for Tamil and English.
- **Voice Interaction**: Voice-to-Text and Text-to-Speech capabilities for hands-free guidance.
- **Device Responsive**: Optimized layout for Mobile, Tablet, and Desktop.
- **Issue Reporting**: Map-integrated reporting with image uploads.
- **Village Portal**: Centralized announcements and government services.
- **Admin CMS**: Dedicated dashboard for managing platform content.

## Technologies
- Backend: Python Flask
- Frontend: HTML5, CSS3, Bootstrap 5, JS
- Database: SQLite

## Setup Instructions

1. Ensure you have Python installed.
2. Clone this project.
3. **API Key Setup**:
   - Create a `.env` file in the root directory.
   - Add your Gemini API Key: `GEMINI_API_KEY=your_key_here`
4. Install dependencies: `pip install -r requirements.txt`
5. Run the application: `python app.py`

7. Access at `http://localhost:5000`
