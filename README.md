# e-Gram Digital Platform

Digital platform that allows villagers to report problems, receive announcements, access government services, and communicate with local authorities.

## Features
- Report Issues with optional image upload
- View Government Schemes & Services
- View Village Announcements
- Admin Dashboard for managing platform content

## Technologies
- Backend: Python Flask
- Frontend: HTML5, CSS3, Bootstrap 5, JS
- Database: SQLite

## Setup Instructions

1. Ensure you have Python installed.
2. Clone or download this project.
3. Open a terminal in the project directory.
4. (Optional) Create a virtual environment: `python -m venv venv` and activate it.
5. Install dependencies: `pip install -r requirements.txt`
6. Run the application: `python app.py`
7. Access at `http://localhost:5000`
